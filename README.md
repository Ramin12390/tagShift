# 🚀 TagShift - Ultimate System Monitor & Optimizer

TagShift is a premium Windows utility designed for power users and gamers who demand maximum performance and a high-tech aesthetic. Built with a futuristic **Cyber-Glass UI**, it provides deep insights into your hardware while offering tools to optimize your system in real-time.

## ✨ Key Features

*   **💎 Cyber-Glass UI:** A stunning, translucent interface with dynamic neon accents.
*   **📊 Hardware Insights:** Detailed information about your CPU, GPU, RAM, Motherboard, and OS.
*   **📈 Real-Time Stats:** Live monitoring of CPU load/temp, storage, and network speeds.
*   **💀 Task Reaper:** Advanced process management—kill stubborn tasks or set process priorities for maximum gaming performance.
*   **🧹 System Purge:** One-click optimization to clear temporary files and flush DNS cache.
*   **🛡️ Admin Mode:** Runs with high-level privileges for the most accurate performance metrics.

## 📥 How to Download

1. Go to the [**Releases**](https://github.com/Ramin12390/tagShift/releases) section on the right side of this page.
2. Download the latest `TagShift-Setup.exe`.
3. Run the installer and grant **Administrator** privileges for full functionality.
